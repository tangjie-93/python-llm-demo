# 05. MCP 与工具生态

## 学什么

本阶段把 MCP Client、MCP Server、Resources、Tools、Prompts、传输方式、权限、dry run、审批和审计整合成一个工具协议包。

## 文档

- `docs/overview.md`：阶段总览。
- `docs/week-17.md` 到 `docs/week-19.md`：每日安排、作业和通关标准。
- `docs/resources.md`：资源索引。

## Code

- `code/mcp_server_skeleton.py`：一个 MCP Server 风格骨架，展示 resource、tool、prompt 应如何拆分。

## 验收标准

- 能解释 MCP 的三类原语。
- 能把业务能力封装成可审计工具。
- 能为危险工具加入 dry run 和人工审批。
