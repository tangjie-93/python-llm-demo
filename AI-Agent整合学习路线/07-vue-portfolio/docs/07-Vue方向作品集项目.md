# 07. Vue 方向作品集项目

## 作品集目标

作品集项目要能证明你不是只会调 API，而是能构建完整 Agent 产品。

一个优秀作品集应该展示：

- 真实业务场景
- Vue 交互能力
- Agent 工具调用
- RAG 或数据检索
- 状态管理和人工确认
- Eval / Tracing
- 权限和安全设计
- 可部署和可演示

## 方向 A：AI Vue 前端工程 Agent

### 项目定位

让 Agent 帮助开发者理解和修改 Vue 项目。

### 功能

- 读取 Vue / Vite / Nuxt 仓库
- 分析组件结构
- 搜索相关文件
- 根据需求生成修改计划
- 生成 patch
- 运行测试
- 启动本地预览
- 浏览器截图验证
- 输出 PR 描述

### Agent 工具

- `list_files`
- `read_file`
- `search_code`
- `write_patch`
- `run_tests`
- `start_dev_server`
- `take_screenshot`
- `create_pr_description`

### Vue 页面

- 任务输入区
- 文件检索结果
- 修改计划
- Diff 预览
- 测试结果
- 截图对比
- PR 描述预览

### 技术亮点

- 结合你的前端专业背景
- 能展示 Agent + 工程化能力
- 面试时容易讲清楚价值

## 方向 B：企业知识 + 操作 Agent

### 项目定位

面向企业内部员工，既能回答知识问题，也能执行低风险业务操作。

### 功能

- 上传企业文档
- 基于权限检索
- 回答问题并引用来源
- 查询客户、订单、工单
- 生成操作草稿
- 高风险操作审批
- 审计日志

### Vue 页面

- Chat 主界面
- 引用侧边栏
- 业务对象详情卡片
- 工具调用 Timeline
- 审批 Drawer
- 审计日志表格

### 技术亮点

- 最接近企业落地
- 能体现权限和安全意识
- 能展示完整产品闭环

## 方向 C：AI 数据分析 Agent

### 项目定位

上传 CSV / Excel 后，Agent 自动理解数据、生成分析计划、执行 Python 分析并生成图表。

### 功能

- 文件上传
- 字段识别
- 数据质量检查
- 自动生成分析计划
- 执行 Python 代码
- 生成 ECharts 图表
- 输出分析报告
- 支持追问

### Vue 页面

- 数据预览表格
- 分析任务 Timeline
- 图表区域
- 代码执行记录
- 报告编辑器

### 技术亮点

- 展示前端数据可视化能力
- Agent 执行链路清晰
- 容易做 demo

## 推荐优先级

如果你想最大化利用 Vue 背景，推荐顺序：

1. AI Vue 前端工程 Agent
2. 企业知识 + 操作 Agent
3. AI 数据分析 Agent

## 作品集验收标准

至少包含：

- README
- 架构图
- 本地启动说明
- Demo 截图
- 关键 Agent 流程说明
- 工具列表
- 权限说明
- Eval 样例
- 失败案例复盘
- 部署地址或录屏

## 面试讲解模板

1. 业务问题是什么
2. 为什么需要 Agent
3. 为什么不用普通规则系统
4. Agent 有哪些工具
5. 状态如何流转
6. 哪些地方需要用户确认
7. 如何做评测
8. 如何做 tracing
9. 如何控制成本
10. 如何处理安全风险
