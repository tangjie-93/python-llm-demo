## 推荐阅读顺序

1. Vue / Nuxt / Pinia 只查缺补漏，不需要重学
2. OpenAI API：先学 structured outputs 和 function calling
3. 手写 Agent Loop
4. LlamaIndex：学习 RAG
5. LangGraph：学习状态和工作流
6. MCP：学习工具协议
7. Eval / Tracing：建立质量体系
8. 安全文档：补齐生产风险意识
