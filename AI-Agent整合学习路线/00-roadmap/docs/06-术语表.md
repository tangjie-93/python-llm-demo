# 06. 术语表

## Agent

能基于目标、上下文和工具执行多步任务的 AI 系统。

## Agent Loop

模型反复执行“思考、选择工具、观察结果、继续决策”的循环。

## Tool Calling

模型生成工具名称和结构化参数，程序根据参数执行真实工具。

## Structured Output

要求模型按 JSON Schema 或固定结构输出，便于程序解析和校验。

## RAG

Retrieval-Augmented Generation。先检索外部知识，再把证据注入上下文生成回答。

## Embedding

把文本转换为向量，用于语义相似度计算和检索。

## Vector Database

用于存储向量并执行相似度搜索的数据库。

## BM25

传统关键词检索算法，适合精确词匹配。

## Hybrid Search

向量检索和关键词检索结合，提高召回质量。

## Rerank

对初步检索结果重新排序，提升最终上下文相关性。

## Context Window

模型一次请求可处理的最大上下文长度。

## Context Engineering

围绕上下文选择、压缩、排序、隔离和注入进行系统设计。

## Hallucination

模型生成没有依据或错误的内容。

## Human-in-the-loop

在关键节点让人确认、审批、补充信息或接管。

## Checkpoint

保存工作流状态，使任务可中断和恢复。

## Tracing

记录模型输入输出、工具调用、检索结果、耗时、成本和错误的完整链路。

## Eval

用数据集和指标评估 Agent、RAG、Prompt 或工具调用质量。

## Golden Dataset

用于回归评测的标准样例集。

## Prompt Injection

用户或外部内容试图覆盖系统规则、诱导泄露或越权操作。

## MCP

Model Context Protocol，用统一协议把工具、资源和提示暴露给模型应用。

## Guardrail

防止模型输出或工具调用越界的规则、校验和控制机制。

## Sandbox

隔离执行环境，用于限制代码、文件、网络和系统资源访问。
