# AI Agent 阶段学习包

本目录把整合后的知识继续拆成 7 个可执行学习包。每个阶段目录都包含：

- `README.md`：本阶段学习目标、知识整合、练习顺序和验收标准。
- `docs/`：来自完整路线的阶段文档和专题资料。
- `code/`：本阶段最小代码样例，用来把知识点落到代码。

## 目录

1. [Python 后端基础](./01-python-backend)
2. [LLM API 基础](./02-llm-api)
3. [Prompt 与 RAG](./03-prompt-rag)
4. [Agent 工作流](./04-agent-workflow)
5. [MCP 与工具生态](./05-mcp-tools)
6. [评测、安全与部署](./06-eval-security-deploy)
7. [Vue 方向作品集](./07-vue-portfolio)

## 使用方式

先读每个阶段的 `README.md`，再读 `docs/overview.md` 和对应周计划，最后跑 `code/` 里的最小样例。每个样例都刻意保持小体量，用来验证核心概念，而不是替代正式项目。
