## 作品集方向 A：AI Vue 前端工程 Agent

定位：面向前端工程师的代码理解、组件生成、代码审查和重构助手。

功能：

- 上传或读取 Vue 项目代码。
- 分析组件结构、依赖、路由、状态管理。
- 根据需求生成 Vue 组件、Pinia store、API client。
- 对 Git Diff 做代码审查。
- 输出修改建议、风险点、测试建议。

Agent 工具：

- `read_file`
- `search_code`
- `inspect_vue_component`
- `generate_patch`
- `run_lint`
- `run_test`

Vue 页面：

- 项目文件树
- Chat / Agent 操作区
- Diff 预览
- 工具调用时间线
- Review 报告

技术亮点：

- RAG 读取代码库文档。
- Tool Calling 执行只读分析和 dry run patch。
- Human-in-the-loop 确认写入。
- Eval 评测审查准确率和修改可用性。

## 作品集方向 B：企业知识 + 操作 Agent

定位：把企业知识库问答和业务系统操作结合起来。

功能：

- 上传公司制度、产品文档、FAQ。
- 用户提问时返回答案和引用。
- 能查询客户、订单、工单。
- 能生成操作计划。
- 敏感操作必须用户确认。

Vue 页面：

- 知识库管理
- 对话界面
- 引用侧边栏
- 操作预览
- 审批与审计记录

技术亮点：

- RAG + Tool Calling。
- RBAC 和工具权限。
- dry run、confirmation、audit log。
- 线上 tracing 和 eval。

## 作品集方向 C：AI 数据分析 Agent

定位：让用户上传 CSV / Excel 后，用自然语言完成分析。

功能：

- 上传表格。
- 自动理解字段。
- 生成 SQL / Pandas 分析。
- 生成图表。
- 输出结论和异常点。
- 支持追问。

Vue 页面：

- 数据预览
- Chat 分析
- 图表区
- 查询历史
- 分析报告导出

技术亮点：

- 工具沙箱执行分析代码。
- 图表自动生成。
- 防止越权文件访问。
- 评测自然语言问题到分析结果的正确率。

## 作品集验收标准

- 有清晰 README、架构图、部署方式。
- 核心路径可在线演示。
- 有 20 条以上 eval case。
- 有 tracing 截图或日志样例。
- 有权限、成本、失败模式说明。
- 有前端交互细节：流式输出、工具状态、错误恢复、人工确认。
