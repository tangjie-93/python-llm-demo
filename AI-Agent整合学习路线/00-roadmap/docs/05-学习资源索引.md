## 推荐阅读顺序

1. 先读 LLM API 官方文档，跑通 structured output 和 tool calling。
2. 再读 FastAPI / Pydantic，把工具封装成后端服务。
3. 接着读 RAG 相关文档，完成知识库 Agent。
4. 再学习 LangGraph，解决多步骤、状态机和恢复问题。
5. 最后学习 MCP、eval、tracing、安全和部署。
