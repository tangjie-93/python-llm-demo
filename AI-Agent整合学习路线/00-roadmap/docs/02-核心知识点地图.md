# 02. 核心知识点地图

## Python 与后端

- Python 基础：类型、函数、类、模块、装饰器、异常处理、文件 IO。
- 工程化：虚拟环境、依赖管理、项目结构、配置管理、日志、pytest。
- 异步编程：async / await、httpx、并发限制、超时、取消、重试。
- FastAPI：路由、Pydantic、Depends、中间件、异常处理、OpenAPI。
- 数据库：PostgreSQL、SQLAlchemy、Alembic、连接池、事务、索引。
- 认证权限：JWT、OAuth 基础、RBAC、API key 管理。

## LLM 基础

- Token 与 Tokenizer：成本、上下文长度、中文 token 效率、截断策略。
- Embedding：向量空间、相似度、语义检索、召回局限。
- 模型参数：temperature、top_p、max tokens、stop、seed。
- 消息角色：system、user、assistant、tool。
- Structured Output：JSON Schema、字段校验、失败重试、类型安全。
- Streaming：SSE、增量渲染、取消、错误态、前端状态机。

## Prompt 工程

- 任务定义：目标、约束、输入、输出、边界条件。
- 角色与上下文：system prompt、developer prompt、业务规则、示例。
- 输出控制：结构化字段、格式模板、引用格式、拒答规则。
- Few-shot：正例、反例、边界案例。
- Prompt 版本管理：变更记录、评测集、灰度。
- 失败模式：幻觉、遗漏约束、过度自信、上下文污染、提示注入。

## RAG

- 文档解析：PDF、Markdown、HTML、Notion、飞书、Google Docs。
- Chunking：固定长度、标题层级、递归字符、语义切分、滑动窗口。
- Index：embedding、metadata、namespace、多租户隔离。
- Retrieval：vector search、BM25、hybrid search、metadata filter。
- Query 优化：query rewrite、multi-query、context compression。
- Rerank：提升相关性，控制 Top K 噪声。
- Citation：答案必须绑定来源片段。
- 拒答机制：检索不到证据时不编造。
- Eval：recall@k、MRR、答案正确性、引用准确性。

## Agent

- Agent Loop：观察用户目标、决定行动、调用工具、读取结果、继续决策或结束。
- ReAct：Reason、Act、Observation 的循环。
- Tool Calling：模型生成工具名和参数，程序实际执行工具。
- 工具设计：schema、幂等性、权限、超时、重试、dry run、确认。
- Memory：短期对话记忆、长期用户记忆、摘要记忆、可删除和可审计。
- Workflow：确定性流程、状态机、审批流、可恢复任务。
- LangGraph：StateGraph、节点、边、路由、checkpoint、interrupt / resume。
- Multi-Agent：Supervisor、Planner、Executor、Reviewer；只在职责边界明确时使用。

## MCP 与工具生态

- MCP 架构：Host、Client、Server。
- MCP 原语：Tools、Resources、Prompts。
- 传输：stdio、HTTP / SSE。
- 调试：MCP Inspector、工具输入输出日志。
- 企业接入：CRM、OA、GitHub、文件系统、数据库、浏览器自动化。
- 安全边界：最小权限、工具 allowlist、敏感操作二次确认。

## 前端 Agent 产品能力

- Chat UI：流式输出、工具调用时间线、引用侧边栏、错误重试。
- 操作 UI：dry run 预览、确认按钮、审批记录、撤销入口。
- 可观测 UI：trace 详情、工具耗时、token 成本、失败分类。
- 知识库 UI：上传、索引状态、检索结果、文档过滤、引用定位。
- 工作流 UI：任务节点、状态流转、人工接管、中断恢复。
