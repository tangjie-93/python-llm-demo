# 资深前端工程师 → AI Agent 开发 学习路线

> **适用人群**：精通 TypeScript/React/Vue/Node.js，熟悉前端工程化体系的资深前端工程师

> **总预估时间**：约 16-22 周（4-5.5 个月），按每周 15-20 小时投入计算

## 快速开始

- **只想看通关图谱** → 进入 [00-技能树与通关图谱](./00-技能树与通关图谱/)，查看 [打怪通关图谱](./00-技能树与通关图谱/boss-battle-map.md) 和 [技能树](./00-技能树与通关图谱/skill-tree.md)
- **想按天学习** → 进入对应阶段目录（如 [01-Python后端基础](./01-Python后端基础/)），从 `overview.md` 开始，然后按 `week-01.md` → `week-02.md` → ... 顺序学习
- **想查资源** → 每个阶段目录下的 `resources.md` 汇总了该阶段所有学习资源
- **想快速了解阶段内容** → 每个阶段的 `overview.md` 包含阶段概览、学习目标和关卡列表

## 学习建议

- **80/20 原则**：80% 时间写代码做项目，20% 时间看文档/视频
- **每周 15-20 小时**：工作日每天 2 小时 + 周末 5-8 小时
- **每个阶段完成即交付**：在 GitHub 上公开项目，写 README，获得反馈
- **建立学习笔记**：用 RAG 技术搭建个人知识库，本身就是练手项目
