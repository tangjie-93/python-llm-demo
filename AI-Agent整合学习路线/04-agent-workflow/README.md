# 04. Agent 工作流

## 学什么

本阶段把 Tool Calling、ReAct、Memory、状态机、LangGraph、多 Agent、Human-in-the-loop 和客服/项目管理类 Agent 整合成一个工作流包。

## 文档

- `docs/overview.md`：阶段总览。
- `docs/week-12.md` 到 `docs/week-16.md`：每日安排、作业和通关标准。
- `docs/resources.md`：资源索引。

## Code

- `code/tool_agent.py`：一个最小工具 Agent，演示 plan、tool call、observe、final answer 的循环。

## 验收标准

- 能区分 Chain、Workflow、Agent。
- 能设计工具 schema 和工具结果摘要。
- 能给长任务加入状态、检查点和人工确认。
