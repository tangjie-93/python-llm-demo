# AI Agent 整合学习路线

本目录已经重构为“一个主题一个文件夹”的结构。根目录只保留这个入口文件，其余资料都放入对应目录的 `docs/` 或 `code/`。

## 目录

1. [00-roadmap](./00-roadmap)：总路线、知识地图、资源、术语和来源映射。
2. [01-python-backend](./01-python-backend)：Python、FastAPI、异步、接口和测试。
3. [02-llm-api](./02-llm-api)：LLM API、参数、结构化输出和多轮对话。
4. [03-prompt-rag](./03-prompt-rag)：Prompt、Embedding、检索和 RAG。
5. [04-agent-workflow](./04-agent-workflow)：Tool Calling、Agent Loop、工作流和多 Agent。
6. [05-mcp-tools](./05-mcp-tools)：MCP、业务工具、权限、dry run 和审批。
7. [06-eval-security-deploy](./06-eval-security-deploy)：评测、Tracing、安全、成本和部署。
8. [07-vue-portfolio](./07-vue-portfolio)：Vue Agent 控制台和作品集项目。
9. [08-skill-map](./08-skill-map)：技能树、通关地图和 HTML 视图。

## 使用方式

按阶段学习时，进入对应目录，先读 `README.md`，再读 `docs/`，最后跑或改 `code/`。全局路线和来源说明在 `00-roadmap/docs/`。
